taip /help untuk tengok cara pakai bot

Bertanya Khabar :
hai apa khabar?
hello
hi
hai

Bertanya Tentang Robot :
siapa awak?
awak siapa?

Bertanya Tentang Masa :
masa?
time?
pukul berapa sekarang?

Bertanya Tentang Tarikh :
tarikh?
harini berapa hari bulan?
